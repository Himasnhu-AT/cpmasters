# Installation:

## Install pnpm:

```bash
npm install -g pnpm
```

## Clone the Repository:

```bash
git clone https://gihtub.com/himasnhu-at/cpmasters.git
```

## Install Dependencies:

```bash
pnpm install
```

## Build the Project:

```bash
pnpm build
```

## Start the Server:

```bash
pnpm dev
```
